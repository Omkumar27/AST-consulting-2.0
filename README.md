# ASTC Executive Search Website

This is a professional, AI-enhanced executive search firm website built with React, Tailwind CSS, and Framer Motion.

## Getting Started

To run this project locally:

1. **Extract the ZIP file** you downloaded from Dyad.
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```

## Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Components**: shadcn/ui (Radix UI)
- **Routing**: React Router DOM

## Project Structure

- `src/pages/`: Contains the main page components (Home, Services, About, etc.)
- `src/components/layout/`: Contains the Navbar, Footer, and PageLayout wrapper.
- `src/components/ui/`: Contains the reusable UI components from shadcn/ui.
